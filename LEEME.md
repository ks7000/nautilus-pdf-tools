## Instalar

### Nautilus

```sh
sudo add-apt-repository ppa:atareao/nautilus-extensions
sudo apt update
sudo apt install nautilus-pdf-tools
nautilus -q
```

### Nemo

```sh
sudo add-apt-repository ppa:atareao/nemo-extensions
sudo apt update
sudo apt install nemo-pdf-tools
nemo -q
```

### Caja

```sh
sudo add-apt-repository ppa:atareao/caja-extensions
sudo apt update
sudo apt install caja-pdf-tools
caja -q
```

### Requerimientos

Dependencias comunes requeridas,

* python
* python-gi
* python-gi-cairo
* python-cairo
* python-pil
* gir1.2-gtk-3.0
* gir1.2-gdkpixbuf-2.0
* gir1.2-poppler-0.18
* python-pypdf2

#### Solo para Nautilus

* gir1.2-nautilus-3.0
* python-nautilus

#### Solo para Nemo

* gir1.2-nemo-3.0
* python-nemo

#### Solo para Caja

* gir1.2-caja-3.0
* python-caja
